Two Lighting Control Methods

ARGB fan refers to addressable and programmable, that is every light-emitting point on the fan can be controlled, and the color changes are very rich, which is more fashionable and dazzling compared with ordinary RGB.

1.Button Control

Press the LED button on the I/O panel to easily switch the ARGB fan light.(Press the LED button twice to return to the original mode)

2.Motherboard Control

Connect the 5V 3Pin ARGB header to motherboard, you can control the fans through the ARGB software of the motherboard.

Musetex motherboard cable only suitable for 5V 3PIN header.(Press the button for three seconds to switch to motherboard mode)

Forbidden try to connect the 12V 4PIN header with the motherboard cable, If your motherboard is a 12V 4PIN header, please only use the button to control the ARGB fans to change the color.

Musetex Fan Control Board

Equipped with Musetex fan control board, please connect 5V 3PIN motherboard interface to control ARGB fan. ( Forbidden connect with 12V 4PIN header)

A1 2PIN LED line interface

A2 Power cord interface

A3 Motherboard line interface
